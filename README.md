# Project Soul Cube
ProjectSoulCube focuses on the application and demonstration of the capabilities of the [ProceduralLevelGenerator](https://github.com/gaganref/ProceduralLevelGenerator) which is level generation tool developed as a part of my dissertation project. The product of this phase is a complete game, leveraging the unique features of the custom tool.

In the game, players navigate through procedural generated levels to find and collect various items. These items are to be offered at a specified location within the game world. This creates an environment with a high level of replayability and diverse gameplay experiences, as each level presents new layouts and challenges.

The primary challenge for the player is to accomplish the goal before their health depletes to zero. The balance of exploration and survival presents a dynamic challenge, engaging players in an exciting race against time while immersing them in the unique, procedural generated landscapes.

You can Download the game at https://gaganref.itch.io/projectsoulcube
